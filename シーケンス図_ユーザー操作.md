# シーケンス図 - ユーザー操作フロー

```mermaid
sequenceDiagram
    participant User as ユーザー
    participant App as Sample01<br/>アプリケーション
    participant Scanner as Scanner<br/>入力処理
    participant Logic as 計算ロジック

    User->>App: アプリケーション起動
    App->>Scanner: 入力待機開始
    
    User->>Scanner: 商品価格（税抜）を入力
    Scanner->>App: 価格データを返却
    
    App->>Scanner: 入力待機開始
    User->>Scanner: カテゴリを入力<br/>(FOOD/ALCOHOL/NEWSPAPER/OTHER)
    Scanner->>App: カテゴリデータを返却
    
    App->>App: カテゴリのバリデーション
    
    alt カテゴリが有効な場合
        App->>Logic: calcPriceWithTax(price, category)
        Logic->>Logic: isReducedTax(category)で<br/>軽減税率対象か判定
        Note over Logic: FOOD or NEWSPAPER<br/>→ 軽減税率(8%)<br/>その他<br/>→ 標準税率(10%)
        Logic->>Logic: 税込価格を計算
        Logic-->>App: 計算結果を返却
        
        App->>User: 税込価格と税率を表示
    else カテゴリが無効な場合
        App->>User: エラーメッセージを表示
        Note over App: "無効なカテゴリです..."
    end
    
    App->>Scanner: Scannerをクローズ
    App->>User: プログラム終了
```

## シーケンス詳細

| ステップ | 処理内容 | 説明 |
|---------|---------|------|
| 1 | アプリケーション起動 | main()メソッドが実行される |
| 2 | 価格入力 | nextDouble()で税抜価格を入力 |
| 3 | カテゴリ入力 | next().toUpperCase()でカテゴリを入力 |
| 4 | バリデーション | Category.valueOf()でカテゴリを検証 |
| 5 | 税率判定 | isReducedTax()で軽減税率対象を判定 |
| 6 | 価格計算 | calcPriceWithTax()で税込価格を計算 |
| 7 | 結果表示 | printf()で結果を表示 |
| 8 | 終了 | Scannerをクローズしてプログラム終了 |
